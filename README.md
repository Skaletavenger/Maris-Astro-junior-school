# Sunrise Academy Website

This is a Next.js school website built as a simple landing page for Sunrise Academy. It includes sections for about, programs, admissions, and contact information.

## Getting Started

Install dependencies if needed:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Project Structure

- `app/page.tsx` — main landing page content
- `app/layout.tsx` — site metadata and global layout
- `app/globals.css` — Tailwind CSS configuration and base styles

## Notes

This project uses Next.js 16 and Tailwind CSS 4.
